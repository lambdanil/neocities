These are some of my more interesting projects.


neocities

Yep, it's this website's code. Horribly written mix of html, css and some javascript.


system config

This is my full system configuration, separated into individual modules. It's fully written in org-mode, and includes my Emacs config.


astOS

An immutable Linux meta-distribution. It's based on Arch Linux, and uses the btrfs filesystem to implement transactional, atomic updates. It is currently unmaintained.


wordle solver

Exactly what it sounds like. A CLI driven wordle solving bot. Extremely small and written in python.

Sadly the dictionary I use isn't perfectly 1:1 with the full wordle dictionary, so it doesn't figure out some words :/.
